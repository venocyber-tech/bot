WhatsApp Automation Bot

Powered by Venocyber Tech & whatsapp-web.js

✅ Bot is running successfully

Setup Instructions:

  1. Check Heroku logs for the QR code: heroku logs --tail --app YOUR_APP_NAME
  2. Scan the QR code with your WhatsApp within 60 seconds
  3. The bot will automatically reconnect after deployment restarts

Features:

  • Auto-reply to messages with intelligent responses
  • Command system (!help, !info, !time, !status)
  • Keyword-based automatic responses
  • 24/7 operation on Heroku
  • Automatic reconnection on failure
  • Admin notifications

Usage:

Send messages to the bot's WhatsApp number with commands or questions like:

  • !help - Show available commands
  • Hello - Get a greeting response
  • What's the price? - Get pricing information
  • !time - Get current time

Note: This bot uses the unofficial WhatsApp Web API. Use responsibly and comply with WhatsApp's terms of service.
